02 Featured Projects
pddl-copilot ↗
SPL-BGU · MIT · v2.0.0
Open-source MCP plugin marketplace

A Claude Code plugin marketplace that lets LLM agents interact with classical-planning tools through natural language. It packages three pip-installable plugins — a solver (Fast Downward & ENHSP), a validator, and a parser — each exposed as a Model Context Protocol server with accompanying agent skills. The servers are portable across Claude Code, Cursor, and Google Antigravity, with a unified installer and a CI gate running static checks, plugin tests, and MCP-protocol smoke tests on every pull request.

pddl-copilot-experiments ↗
SPL-BGU · research
Evaluation framework for LLM tool use

An evaluation framework that measures whether language models plan better when given tools. It runs LLMs over five PDDL tasks — solve, validate domain / problem / plan, and simulate — in two conditions: with MCP planning tools and without. Outputs are scored against planner-generated ground truth and aggregated with Wilson 95% confidence intervals. The harness supports resumable runs, fast-feedback partial slices, and large model sweeps submitted as SLURM job arrays on a GPU cluster — the empirical companion to the "Toward PDDL Planning Copilot" research line.

Python LLM evaluation MCP Ollama SLURM / HPC Statistical analysis
online_model_learning ↗
information-gain-AML
Active learning of planning models

A research system that learns PDDL action models online — actively choosing which action to execute next by expected information gain, then inferring the underlying model from observed state transitions. The inference step is powered by a custom SAT-based solver doing concurrent constraint search, which substantially cuts the runtime of an otherwise exponential combinatorial problem.
